## 📋 سير عمل Notion — القواعد الأساسية

### أعمدة الحالة (بالترتيب)

| الحالة | الوصف |
|--------|-------|
| 📝 **To Do** | مهمة للقيام بها |
| 🔄 **In Progress** | مهمة قيد التنفيذ |
| 👁️ **Review** | في انتظار مراجعة المدير |
| ✅ **Done** | منتهية (المدير فقط) |
| 🚫 **Blocked** | محظورة — اقرأ التعليق |

### قواعد سير العمل

1. **بدء مهمة**: انقل من `To Do` ← `In Progress`
2. **إنهاء مهمة**: انقل من `In Progress` ← `Review`
3. **⚠️ لا تضع أبداً Done** — المدير فقط يمكنه ذلك
4. **إذا كانت Blocked**:
   - اقرأ تعليق المدير
   - قم بالتصحيحات
   - أعدها إلى `In Progress`
   - بعد الإصلاح، أعدها إلى `Review`

### سير العمل اليومي

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   To Do     │ ──► │ In Progress │ ──► │   Review    │
└─────────────┘     └─────────────┘     └─────────────┘
                           ▲                   │
                           │                   ▼
                    ┌──────┴──────┐     ┌─────────────┐
                    │   Blocked   │ ◄── │  (المدير)   │
                    └─────────────┘     └─────────────┘
```
