## 📋 Workflow Notion — Règles Essentielles

### Colonnes de Statut (dans l'ordre)

| Statut | Description |
|--------|-------------|
| 📝 **To Do** | Tâche à faire |
| 🔄 **In Progress** | Tâche en cours |
| 👁️ **Review** | En attente de révision par le manager |
| ✅ **Done** | Terminé (seulement par le manager) |
| 🚫 **Blocked** | Bloqué — lire le commentaire |

### Règles du Workflow

1. **Commencer une tâche** : Déplacez de `To Do` → `In Progress`
2. **Terminer une tâche** : Déplacez de `In Progress` → `Review`
3. **⚠️ NE JAMAIS marquer comme Done** — Seul le manager peut le faire
4. **Si Blocked** :
   - Lisez le commentaire du manager
   - Faites les corrections
   - Remettez en `In Progress`
   - Une fois corrigé, remettez en `Review`

### Workflow Quotidien

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   To Do     │ ──► │ In Progress │ ──► │   Review    │
└─────────────┘     └─────────────┘     └─────────────┘
                           ▲                   │
                           │                   ▼
                    ┌──────┴──────┐     ┌─────────────┐
                    │   Blocked   │ ◄── │ (Manager)   │
                    └─────────────┘     └─────────────┘
```
